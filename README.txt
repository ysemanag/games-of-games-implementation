================================================================================
                     FILE DEFINITIONS & CONFIGURATION
================================================================================


Clone this repo or Download and unzip GameOfGames_player_mode.zip

Key classes:
PlayGames.java - Runs all of the games that are part of Game of Games.
GetInput.java - Assists in getting all required inputs from the users for all the other classes.

NORMAL VERSION        | TEST VERSION              | BUG VERSION
coin_flip.java        | coin_flip_test.java       | coin_flip_bug.java          Plays Coin Flip.
evenorodd.java        | evenorodd_test.java       | evenorodd_bug.java          Plays Even or Odd.
FindTheRedThread.java | FindTheRedThreadTest.java | FindTheRedThreadBug.java    Plays Find the Red Thread.
GuessTheNumber.java   | GuessTheNumber_test.java  | GuessTheNumber_bug.java     Plays Guess the Number.
Thimble.java          | ThimbleTest.java          | ThimbleBug.java             Plays Find the Thimble.
